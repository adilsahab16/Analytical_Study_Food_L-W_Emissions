# Food loss & Emissions - Exploratory Analysis Using Python

This is a exploratory analysis of global food loss and food system GHG emissions in different regions using Python (Pandas and matplotlib), which examines how dominant trends have evolved over the last 30 years and the factors that affect these trends. Please check the write up [here](https://docs.google.com/document/d/1PyNaTEKX0RZd85M2WnR4LdbSpuRRuu3Kf91zYRdwaQ8/edit?tab=t.0) for more information on data sources, findings and analysis.
